# Graph Reducer
The graph reducer implements app functionality based on the action definitions in the graph-actions.js file.

## Associated Actions
- ** setFocusedTrackId: ** Updates state with the user selected track which is displayed in detail below the graph, i.e. "Mission and Vision" or "Adaptibility"
- ** handleTrackMilestoneChange: ** Updates changes in a given milestone and eligible titles, i.e. total points, sub categories, selected title.
- ** setTitle: ** Sets state with a new title in the eligible job titles.
- ** NameInputChange: ** Sets state with a new user name.

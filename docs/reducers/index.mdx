# Index
---
## ```./src/reducers/index.js```
## Description
The reducers index.js file combines the reducing functions from the various reducing files into a single function that can be passed to the redux store.

## Code Snippet
```
import { combineReducers } from 'redux'
import auth from './auth-reducer'
import graph from './graph-reducer'
import competencies from './competencies-reducer'
import users from './users-reducer'

const reducers = combineReducers({ graph, auth, competencies, users })

export default reducers

```
